# coke-config

## what it is:
a collection of termux configuration files.

## how it works:
the main shell configuration (zsh/zshell) is split into 11 seperate configuration files, the main loader is ~/.zshrc, the shell "modules" are stored in ~/zsh.d.

------------------------------------------

## highly reccomended android apps to have before using coke-config:
1. [Shizuku](https://github.com/RikkaApps/Shizuku)
2. [Unexpected Keyboard](https://github.com/Julow/Unexpected-Keyboard)
